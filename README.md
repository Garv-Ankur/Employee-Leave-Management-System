# Employee Leave Management System

A web-based application to manage employee leave requests efficiently.

## 🔧 Tech Stack
- HTML, CSS, Bootstrap
- JavaScript
- Node.js
- MongoDB

## ✨ Features
- Employee login & dashboard
- Apply for leave
- HR approval/rejection system
- Leave history tracking

## ▶️ How to Run
1. Clone the repo
2. Run `npm install`
3. Start server: `node app.js`
4. Open in browser
